# Helicopter Swivels Website

This repo contains multiple design options for the Helicopter Swivels website.

## Structure

```
option-1/    - Design option 1
option-2/    - Design option 2
option-3/    - Design option 3
assets/      - Shared images and files
company/     - Company info, content, and branding
```

## Adding Company Content

Drop files into the `company/` folder:
- `about.txt` — company description, history, what you do
- `services.txt` — list of products/services
- `contact.txt` — address, phone, email
- `assets/` — logo, product photos, any images
